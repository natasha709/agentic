# Email Troubleshooting Guide

## Common Email Issues

### 1. Can't Send/Receive Emails
**Symptoms:** Emails stuck in outbox, no new emails arriving

**Troubleshooting Steps:**
1. Check internet connection
2. Verify email server settings:
   - Incoming (IMAP): imap.company.com:993
   - Outgoing (SMTP): smtp.company.com:587
3. Verify credentials are correct
4. Check spam/junk folders
5. Verify mailbox isn't full

### 2. Email Not Syncing
**Symptoms:** Old emails only, can't see new emails on mobile

**Solutions:**
1. Force sync on device
2. Remove and re-add account
3. Check sync settings
4. Verify date/time is correct on device

### 3. Password Issues
**Symptoms:** "Invalid credentials" error

**Solutions:**
1. Reset email password via webmail
2. Update password on all devices
3. Check for account lockout

### 4. Attachment Problems
**Symptoms:** Can't open/send attachments

**Solutions:**
- Check file size (max 25MB typically)
- Verify file type isn't blocked
- Try compressing large files

## Email Security

- Never share credentials
- Be wary of phishing emails
- Report suspicious messages
- Use company email for work only
